## What I'm Building Right Now

### Cloud Governance Baseline - Active Development

A hands-on GRC engineering project that translates cloud control requirements into secure Terraform, policy-as-code checks, automated tests, CI/CD enforcement, and audit-ready evidence.

Why it matters: Cloud governance should not depend only on policy documents, spreadsheets, screenshots, and point-in-time reviews. This project demonstrates how control requirements can be embedded into infrastructure delivery and tested before deployment.
